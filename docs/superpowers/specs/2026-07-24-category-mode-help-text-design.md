# 单位类别模式说明文本设计

## 目标

让炮兵和战争机器的三档规则按钮在点击后，立即在按钮下方显示当前档位的详细作用说明。

## 交互

- 每个规则组保留“仅血量”“50%规模”“100%规模”三个互斥按钮。
- 说明文本只显示当前选中档位，切换按钮后即时更新。
- 炮兵和战争机器两个规则组各自维护说明，不互相影响。
- 未保存或缺少旧设置时，默认选中“100%规模”，并显示对应说明。

## 文本内容

- 仅血量：不增加模型数量，只按全局倍率提高该类单位总血量。
- 50%规模：模型数量只提高全局倍率增量的一半，但总血量仍按完整全局倍率计算。
- 100%规模：模型数量按完整全局倍率提高，总血量同步按完整全局倍率增长。

三档说明接入现有中英文及其他支持语言目录，并继续使用现有设置卡片样式。

## 验证

前端测试覆盖默认说明、三个按钮的选中状态和点击后说明切换；生产构建必须通过。
