# 西班牙语支持设计

## 目标

将西班牙语（`es-ES`）作为第六个内置语言加入应用，并让其与现有五种语言享有相同的行为。

## 范围

- 前端提供完整的西班牙语界面文案，不以英语作为正常显示回退。
- 首次启动时将西班牙语系统区域设置识别为 `es-ES`。
- Steam 创意工坊查询与发布使用 Steam 的 `spanish` 语言代码。
- AI 模组翻译提示可将西班牙语作为目标语言。
- 西班牙语更新日志覆盖所有现有版本，并在 0.7.0 中说明新增西班牙语支持。
- 中文和英文 README 均说明西班牙语已受支持。

## 设计选择

采用单独的西班牙语文案目录，而不是给现有位置数组追加第六列。现有五语言数组保持稳定，西班牙语目录通过相同键集覆盖；缺少键会在测试中失败。这避免了对五百余条既有数组逐行改写，同时让新增语言的完整性可直接校验。

## 验证

- 前端测试验证六种语言都有每个界面键的非空译文，并检查西班牙语关键文案。
- Python 测试验证设置保存、系统区域识别、Steam 语言映射、AI 目标语言和全部西班牙语更新日志。
- 运行前端测试、后端相关测试、编译检查与差异空白检查。
